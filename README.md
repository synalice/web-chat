# web-chat
This is a small and very simple online chat that works by sending and getting API requests  
https://syn-server.tech/ 

Goals:
- [x] Basic UI
- [x] API to publish posts
- [x] API to retrieve posts
- [x] Database
- [x] docker-compose
- [ ] Websockets
- [ ] Spam protection
- [ ] Browser notifications
- [ ] Registration
- [ ] 0Auth
- [ ] File upload
- [ ] Admin tools
